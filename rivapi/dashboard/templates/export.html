Backup

Bundles everything this dashboard configures — broadcast/Icecast/ Liquidsoap settings, patchbay routing, install mode, and scheduled tasks — plus Stereo Tool's own device settings and saved presets, into one file. Useful for migrating to a new box or recovering after a catastrophic failure.

Contains real secrets — Icecast source/relay/admin passwords, and a remote MySQL password if this station is in client mode. Handle an exported file the same as you would /etc/rd.conf: not something to share or store casually.
