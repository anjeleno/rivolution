# Wiki mirror

A read-only snapshot of the [GitHub wiki](https://github.com/anjeleno/rivolution/wiki)
(`rivolution.wiki`, a separate repo), kept here so anyone who clones
`rivolution` gets the documentation with it, not just whoever happens
to browse the wiki directly.

**Edit the wiki, not this folder.** These files are copied over
whenever the wiki changes; a change made only here will be overwritten
the next time this mirror is refreshed.

Internal `[[Page Name|slug]]` links are wiki-specific syntax and won't
resolve when reading these files outside the wiki itself.
